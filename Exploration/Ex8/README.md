# 회고(참고 링크 및 코드 개선)
트랜스포머 아키텍처를 활용한 한국어 챗봇 구현 과정을 전반적으로 복습할 수 있었습니다. 특히 영어로 정리된 방대한 이론 자료들을 보며 모델의 세부 동작 원리를 다시 한번 리마인드하는 계기가 되었습니다. 전반적으로 노드의 흐름을 잘 충실히 따랐으나, 모델의 최종 답변에서 문맥적 연관성이 다소 떨어지는 부분이 발견되었습니다. 복잡한 딥러닝 모델일수록 이론적 배경만큼이나 실제 출력되는 결과값에 대한 세밀한 튜닝이 중요하다는 것을 느꼈습니다. 고생 많으셨습니다!
